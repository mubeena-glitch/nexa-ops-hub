# NEXA Web Team Operations Hub

Single-file internal operations dashboard.

## Deploy to Vercel

### Option 1: Drag & Drop (easiest)
1. Go to https://vercel.com/new
2. Drag the entire `nexa-deploy` folder onto the page
3. Click Deploy — done in ~30 seconds

### Option 2: Vercel CLI
```bash
npm i -g vercel
vercel login
vercel --prod
```

### Option 3: GitHub
1. Push this folder to a GitHub repo
2. Import the repo at https://vercel.com/new
3. Vercel auto-detects static site, click Deploy
